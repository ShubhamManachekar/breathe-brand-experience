## Design System Quick Links
-   **Typography:** `font-display` (Fraunces), `font-body` (Manrope).
-   **Colors:**
    -   **Primary:** Navy (`bg-primary`)
    -   **Accent:** Sage/Gold (`text-accent`)
    -   **Background:** Warm White (`bg-background`)
-   **Components:**
    -   `NeoHero`: Shared hero component with variants.
    -   `AnimatedSection`: Reveal animations on scroll.
    -   `PageMeta`: Dynamic SEO headers.
