# Business Solutions (Overview)

## Overview
A high-level directory of industry-specific scent marketing solutions.

**Route:** `/business/solutions`

---

## Layout Structure

### 1. Neo Hero
-   **Headline:** "Where architecture meets atmosphere."
-   **Visual:** Smoke texture.

### 2. Solutions Grid
-   **Content:** Hospitality, Retail, Corporate, Wellness.
-   **Card Style:** `card-loom` (Clean, bordered).
-   **Details:**
    -   **Metric Pill:** e.g., "+35% Guest Satisfaction".
    -   **Highlights:** List of 3 key benefits with spark icons.
-   **Action:** "Learn more" button links to vertical pages.

### 3. Process Steps
-   **Content:** 4-step linear process (Discovery -> Design -> Deploy -> Optimize).
-   **Visual:** Numbered cards.

### 4. CTA Banner
-   **Style:** Glassmorphic rounded container.
-   **Action:** "Request a demo".

---

## Interactions
-   **Card Hover:** Subtle lift and border darken.
