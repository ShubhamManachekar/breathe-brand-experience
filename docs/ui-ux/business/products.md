# Business Products (Listing)

## Overview
A showcase of commercial-grade scent diffusion systems. Key features include HVAC integration, 360-degree views, and bulk quote requests.

**Route:** `/business/products`

---

## Layout Structure

### 1. Neo Hero
-   **Headline:** "Precision scenting for enterprise spaces."
-   **Visual:** Commercial hardware imagery.

### 2. System Highlights
-   **Style:** Row of small cards (`card-clay-sm`).
-   **Content:** Wi-Fi Control, Bluetooth 5.0, HVAC Ready, 2-Year Warranty.

### 3. 360 Explorer (Interactive)
-   **Left:** Rotatable product image (Pro Diffuser).
    -   **Controls:** Rotate Left/Right buttons.
    -   **Hotspots:** Clickable markers revealing tooltips (Nozzle, Control Panel, Cartridge).
-   **Right:** Info cards updating based on selected hotspot.

### 4. Product Range
-   **Layout:** 3-column grid.
-   **Card:**
    -   **Visual:** Product image in a "sunken" container.
    -   **Info:** Name, Model ID, Coverage, Feature list (Checkmarks).
    -   **Actions:** "Request Quote" (Primary), "View in Detail" (Outline).

### 5. CTA Banner
-   **Content:** "Design a scent program built for scale."
-   **Action:** "Talk to a strategist".

---

## Interactions
-   **360 View:** Image source changes based on rotation state.
-   **Hotspots:** Click to toggle info card content.
-   **Request Quote:** Pre-fills contact form with product ID.
