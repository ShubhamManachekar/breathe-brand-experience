This repository contains a Vite + React + TypeScript starter for the Breathe brand experience demo. It includes a site shell, Tailwind & shadcn-ui components, and a local rule-based chatbot for FAQ and scent quotes.

## Quick local development

Install dependencies and run the dev server:

```powershell
npm install
npm run dev
```

Build for production:

```powershell
npm run build
```

## What technologies are used?

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
